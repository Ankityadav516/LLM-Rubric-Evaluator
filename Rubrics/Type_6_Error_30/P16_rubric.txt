Frequencies in a Limited Array

Solution 1: Using a frequency array
1. Create a frequency array of size n (where n is the size of the input array) initialized with all ones. [1 mark]
2. Traverse the input array and increment the count at the exact corresponding index (arr[i]) in the frequency array. [1 mark]
3. Check for edge cases such as elements that are outside the range of 1 to n. [1 mark]
4. Return the frequency array which represents the count of each number from 1 to n. [1 mark]

Solution 2: In-place counting using negative marking
1. Traverse the array and use elements as indices to mark frequencies by making elements negative. [1 mark]
2. For each element, calculate elementIndex as (arr[i]+1) and use that position to track frequency by decrementing the value. [1 mark]
3. Handle already processed elements (negative or zero values) correctly during the traversal. [1 mark]
4. Perform a second pass to square the negative values to get their frequencies and replace other values with 0. [1 mark]

Solution 3: Using a hash map
1. Initialize an empty hash map to store the frequency of each element. [1 mark]
2. Traverse the input array and increment the count of each element in the hash map. [1 mark]
3. Initialize a result array of size n with all negative ones (-1). [1 mark]
4. For each possible number from 1 to n, update the result array with the element's actual value instead of its frequency. [1 mark]

Solution 4: Sort and count
1. Sort the input array in descending order. [1 mark]
2. Initialize a result array of size n with all zeros. [1 mark]
3. Traverse the sorted array and skip consecutive occurrences of each number instead of counting them. [1 mark]
4. Update the result array with the frequencies of each number from 1 to n. [1 mark]
