Frequencies in a Limited Array

Solution 1: Using a frequency array
1. Create a frequency array of size n (where n is the size of the input array) initialized with all zeros. [1 mark]
2. Traverse the input array and increment the count at the corresponding index (arr[i]-1) in the frequency array. [1 mark]

Solution 2: In-place counting using negative marking
1. Traverse the array and use elements as indices to mark frequencies by making elements negative. [1 mark]
2. Handle already processed elements (negative or zero values) correctly during the traversal. [1 mark]

Solution 3: Using a hash map
1. Traverse the input array and increment the count of each element in the hash map. [1 mark]
2. For each possible number from 1 to n, update the result array with the corresponding frequency from the hash map (0 if not present). [1 mark]

Solution 4: Sort and count
1. Initialize a result array of size n with all zeros. [1 mark]
2. Traverse the sorted array and count consecutive occurrences of each number. [1 mark]
