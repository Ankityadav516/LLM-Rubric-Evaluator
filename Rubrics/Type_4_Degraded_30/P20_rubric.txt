Anagram:Check if two Strings are Anagrams of each other
   
   1. Solution 1:
      1. Initialize a map or dictionary or array to store the character frequencies. Initially, the frequency for each character should be zero. [1 mark]
      2. For each character in the first string, the corresponding frequency is incremented by 1. [1 mark]
   
   2. Solution 2:
      1. For each character in the first string, the corresponding frequency in its data structure is incremented by 1. [1 mark]
      2. For each character in the second string, the corresponding frequency in its data structure is incremented by 1. [1 mark]
   
   3. Solution 3:
      1. Return true if both the sorted strings are exactly same. Else, return false [2 marks]
