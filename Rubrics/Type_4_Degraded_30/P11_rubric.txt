Solution 3: Bottom-Up DP (Tabulation)
1. Allocate a 2D boolean array of size (n+1)×(sum+1) [1 mark]
2. Initialize first column (sum=0) to true for all rows [1 mark]
3. Process elements in sequence from 1 to n [1 mark]
4. When current element value > current sum: dp[i][j] = dp[i-1][j] [2 marks]
5. Otherwise: dp[i][j] = dp[i-1][j] OR dp[i-1][j-arr[i-1]] [2 marks]
6. Return the final result from the correct cell dp[n][sum] [1 mark]
